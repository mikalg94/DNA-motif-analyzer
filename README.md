# Analizator motywów DNA

Aplikacja napisana w języku Python służąca do analizy sekwencji DNA i wykrywania motywów sekwencyjnych.

## Funkcje
- Wczytywanie sekwencji DNA z plików FASTA lub TXT
- Wykrywanie motywów
- Podstawowa analiza statystyczna
- Wizualizacja rozkładu motywów
- Interfejs graficzny

## Technologie
Python, NumPy, Pandas, Matplotlib, Biopython, Tkinter
